# authorization-server

authorization-server spring boot 3


#### 使用说明

1.  jdk version >= 1.7
2.  spring boot version 3.1.0
3.  mysql version > 8.0
4.  数据初始化 [authorization_server_db.sql] 
5.  测试客户端client id：**client**，测试客户端secret：**secret**
6.  测试用户账号：**user/password**, **haichen/123456**
